# GhalghayTools — Платформа инструментов для ингушского языка и истории

## Общая идея

Набор независимых веб-приложений, объединённых единым UX: общая шапка с лаунчером,
общий стиль, единый домен. Пользователь переходит между инструментами как между
приложениями Google Workspace — не замечая, что это разные сервисы.

---

## Архитектура

### Микросервисы

Каждый модуль — отдельный Spring Boot сервис со своей БД, деплоем и портом.
Нет общего состояния, нет общих зависимостей. Связь только через UI.

```
ghalghay.tools/            ← landing + лаунчер (статика)
ghalghay.tools/spell/      ← spell-checker (Spring Boot 4 / Java 25)
ghalghay.tools/map/        ← history-map   (Spring Boot 3 / Java 21)
ghalghay.tools/learn/      ← (будущий) тренажёр языка
ghalghay.tools/dict/       ← (будущий) полный словарь
```

Или через субдомены:
```
spell.ghalghay.tools
map.ghalghay.tools
learn.ghalghay.tools
```

## App Launcher — «кнопка Google»

### Идея

В правом верхнем углу каждого приложения — иконка сетки (⋮⋮⋮ / ⠿),
по клику открывается панель со всеми приложениями платформы.
Как Google Apps, Notion, Atlassian.

```

│  ГIалгIай мотт — Орфограф          [⠿] [👤]│  ← шапка любого приложения
└────────────────────────────────────────────┘
                                      │
                              ┌───────▼────────────┐
                              │  GhalghayTools      │
                              │  ┌──────┐ ┌──────┐  │
                              │  │  АА  │ │  🗺  │  │
                              │  │Орфо- │ │Карта │  │
                              │  │граф  │ │истор.│  │
                              │  └──────┘ └──────┘  │
                              │  ┌──────┐ ┌──────┐  │
                              │  │  📚  │ │  +   │  │

```

### Реализация

**Вариант A — общий npm-пакет (рекомендуется)**
- `shared/launcher/` — отдельный React-компонент в монорепо
- Каждый фронтенд устанавливает его как локальную зависимость
- Конфиг приложений: один JSON-файл `apps.json` в `shared/`

**Вариант B — Web Component**
- Один скрипт `launcher.js` подключается через `<script>` тегом в каждом `index.html`
- Работает с любым фреймворком и без него
- Проще деплоить: достаточно обновить один файл на CDN

**Вариант C — копипаст (быстрый старт)**
- Скопировать компонент в каждый модуль, обновлять руками
- Не масштабируется, но можно сделать прямо сейчас

## Общие ресурсы (shared/)

Планируется папка `shared/` в корне монорепо:

```
shared/
├── launcher/        ← App Launcher компонент
│   ├── AppLauncher.jsx
│   ├── AppLauncher.css
│   └── apps.json    ← конфиг приложений
├── theme/           ← общие CSS-переменные, цвета, шрифты
│   └── variables.css
└── icons/           ← логотипы и иконки платформы
```

## Деплой

- Каждый сервис деплоится независимо (Railway / Fly.io)
- Роутинг по пути (`/spell/`, `/map/`) — через nginx reverse proxy или Railway custom domains
- Первый шаг: кастомный домен `ghalghay.tools` с редиректами на текущие Railway URL
